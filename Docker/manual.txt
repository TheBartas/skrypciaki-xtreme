Wymagania to zainstalowany docker compose.
0. wchodzicie do folderu plusFlix, z niego wywołujecie wszystkie komendy
1. docker compose build, do zbudowania obrazu i robienia aktualizacji pakietów
2. docker compose up, do uruchamiania kontenera, aplikacja jest dostępna na 127.0.0.1:8080
3. docker compose down, do wyłączenia kontenera
4. docker compose exec plusflix bash, jeśli chcecie wejść do kontenera, stamtąd możecie wywoływać
   wszystkie komendy symfony cli:
    php bin/console debug:router

6. Migracja do bazy danych: php bin/console doctrine:migrations:migrate
